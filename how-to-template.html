Mastering the Template Editor

The comprehensive guide to building dynamic, professional templates in MaxCredible.

Section 1

The Basics: WYSIWYG Editing

The editor is "What You See Is What You Get." It works just like Word or Google Docs for basic tasks.

  • Click and Type: Simply click anywhere in the white space to start writing text.
  • Formatting: Use the toolbar icons to style your text (Bold, Italic, Underline).
  • Alignment: Use the alignment buttons to center text, align left, or align right.
  • Undo/Redo: Made a mistake? Press Ctrl+Z (Cmd+Z) immediately.
Section 2

The Golden Rule of Email Design

Building templates for emails is different from web design. Email clients (Outlook, Gmail) are old-fashioned.

The Golden Rule: Use Tables for Layout
Never use "columns", "float", or "flexbox" to position items side-by-side. Always use an HTML Table with invisible borders.
Section 3

Working with Tables

Tables are your grid system. You can manipulate them visually or via code.

Visual Editing

  1. Insert Table: Click the Grid icon in the toolbar.
  2. Right-Click Menu: Right-click inside any cell to add rows or merge cells.
  3. Invisible Borders: Right-click > Table Properties > Set Border width to 0.

Source Code Editing

For pixel-perfect control, click the <> button.

<table width="100%" border="0"> <tr> <!-- Row --> <td width="50%">Left Content</td> <td width="50%">Right Content</td> </tr> </table>
Section 4

Hybrid Templates

Create ONE template for both Email and Print using conditional tags.

The Email View

Wraps digital content like payment buttons:

[BeginDeleteIfLetter]
HTML Email Code
[EndDeleteIfLetter]

The Print View

Wraps formal letter layout and addresses:

[BeginDeleteIfEmail]
HTML Print Code
[EndDeleteIfEmail]
Section 5

Variables & Placeholders

Common placeholders to inject real data:

  • [PackageCustomerName] // Debtor Name
  • [InvoiceNumber] // Invoice ID
  • [InvoiceAmount] // Amount

The Repeater Loop

To show multiple invoices, wrap the row in repeaters:

Start: <th>... [BeginInvoiceRepeat]</th>
End: <td>... [EndInvoiceRepeat]</td>
Section 6

Pro Tips & Tricks

Images are Links

Images must be hosted online. You cannot Paste an image from your desktop.

Always Preview

Use the Preview button with a real debtor to test your table loops.

Clean Paste

Use "Paste as Text" (Ctrl+Shift+V) to avoid broken Word formatting.

Currency Symbols

Always add [PackageCurrencyShort] before amounts.

Payment Links

Turn the text [PackagePaymentLink] into a clickable button.

AI Troubleshooting

Paste your source code into ChatGPT to fix broken HTML syntax instantly.
